flux is a note-taking app.

**Features**
Authentication
Separate Organisation
Themes [Light / Dark]
Real-Time Boards
Drawing & Shapes Support
Preview Actions of your friends
Add Text

Check it out here: [flux](https://flux.stormej.me/)
